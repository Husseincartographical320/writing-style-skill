## 🧭 What This Tool Does

writing-style-skill helps you build and refine a writing style file called `SKILL.md`.

A simple flow guides the work:

- AI writes a first draft
- You edit the draft
- The app learns from your edits
- It extracts rules from the changes
- `SKILL.md` improves over time

This makes it useful for teams, solo writers, and anyone who wants steady style control in one place.

## 🔁 Auto-Learning Flow

The app follows a simple pattern:

1. AI creates a draft
2. You fix the draft
3. The app compares the old and new text
4. It finds style patterns in your edits
5. It adds those patterns to `SKILL.md`

Over time, this gives you a style file that matches how you write.

## 🎯 Best Results

Use short edits when you want clear rule extraction. Focus on changes in:

- tone
- word choice
- sentence length
- formatting
- heading style
- list style

If you edit a lot at once, the app may pull in more rules than you want. Small edits make it easier to see what changed.
